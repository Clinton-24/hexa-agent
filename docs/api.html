API Reference

Complete Hexacrimson API for agent management, configuration, metrics streaming, and billing.

Quickstart API Reference Security CLI Reference

Base URL

https://YOUR_HOST

Health

GET /health

Returns control-plane health and version.

Fleet status

GET /api/status

High-level counters: active agents, installing, pending, environments, uptime.

List agents

GET /api/agents

Enroll agent

POST /api/agents/enroll
Content-Type: application/json
X-Enroll-Token: hxl-...

{
  "token": "hxl-7f3a9c2e1b4d8a6f",
  "hostname": "edge-01",
  "environment": "aws",
  "region": "us-east-1"
}

Returns agent id, pid, and connection details.

Heartbeat

POST /api/agents/:id/heartbeat
Content-Type: application/json

{ "metrics": { "cpu": 12.4, "memory": 41.2 } }

Modules

POST /api/agents/:id/modules

{ "monitoring": true, "threatDetection": true, "wifiIntelligence": false }

Remove agent

DELETE /api/agents/:id

Metrics summary

GET /api/metrics/summary

Events

GET /api/events

Billing

GET  /api/billing/subscription
GET  /api/billing/invoices
POST /api/billing/subscribe
{
  "plan": "Pro",
  "price": 99,
  "period": "monthly",
  "addonSecurity": true,
  "method": "card"
}
