Quickstart Guide

⏱ ~8 min readAgent v3.2.1

Get your first Hexacrimson agent up and running in under 5 minutes. Covers installation, authentication, and basic commands.

Quickstart API Reference Security CLI Reference

1. Download

Run the one-liner installer. It auto-detects your OS and architecture, downloads the signed agent binary, verifies the checksum, and enrolls with your control plane.

curl -sSL https://YOUR_HOST/install.sh | bash

Windows (PowerShell):

irm https://YOUR_HOST/public/agent.ps1 | iex

2. Authenticate

The agent generates a unique identity and registers using a short-lived enrollment token. Demo token (override with HXL_TOKEN):

export HXL_TOKEN=hxl-7f3a9c2e1b4d8a6f
curl -sSL https://YOUR_HOST/install.sh | bash

3. Manage

Once connected, manage agents from the dashboard — push module configs, run heartbeats, and view metrics in real time.

open https://YOUR_HOST/dashboard/
  • View fleet CPU / memory / threat stats
  • Enable Monitoring, Threat Detection, or WiFi Intelligence modules
  • Decommission agents you no longer need
