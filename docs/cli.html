CLI Reference

Commands and flags for the Hexacrimson agent CLI and HTTP equivalents used by this control plane.

Quickstart API Reference Security CLI Reference

Install

curl -sSL https://YOUR_HOST/install.sh | bash

agent enroll

agent enroll --token hxl-7f3a9c2e1b4d8a6f [--env aws] [--region us-east-1]

HTTP equivalent:

curl -X POST /api/agents/enroll \
  -H "Content-Type: application/json" \
  -H "X-Enroll-Token: hxl-7f3a9c2e1b4d8a6f" \
  -d '{"token":"hxl-7f3a9c2e1b4d8a6f","hostname":"edge-01"}'

agent status

agent status
agent status --id hxl-a1b2c3d4

HTTP: GET /api/agents or GET /api/agents/:id

agent heartbeat

agent heartbeat --id hxl-a1b2c3d4

HTTP: POST /api/agents/:id/heartbeat

Modules

agent module enable threat
agent module enable wifi
agent module enable monitoring

HTTP: POST /api/agents/:id/modules
